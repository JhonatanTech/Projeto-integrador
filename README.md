# Projeto-integrador
Projeto destinado ao Projeto Integrador e TG da Faculdade de Tecnologia de São Paulo no curso de Gestão da Tecnologia da Informação na Unidade de Franco da Rocha <br><br>
Este Projeto esta destinado a resolver uma problemática localizada na Escola Estadual Benedito fagundes Marques, localizada no centro da cidade.<br><br>
A proposta é um software para o gerenciamento do estoque/almoxarifado.

## Tecnologias usadas
### Front-end
* Javascript
* CSS
* HTML
* jQuery
* Materialize
* Mask-plugin jQuery
### Back-end
* PHP
* Mysql
